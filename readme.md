## Features 项目特性

- [ ] 插件系统，便携地在原有基础上拓展功能
- [ ] 模板引擎，更快速地开发高质量的页面
- [ ] 采用微服务架构，提高了v1 版本服务端的扩展性和服务稳定性
- [ ] 统一的社区规范，提高了开发者的友好度
- [ ] 项目核心将永远开源
